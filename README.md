# 🐍 Python Learning Journey

> *25 programs in 19 days — from `print("Hello World")` to a working Weather API app*

Welcome to my hands‑on Python learning repository. This is not a theoretical course. It’s a **build‑first, commit‑by‑commit** record of my journey from absolute beginner to writing real‑world scripts and GUI applications.

If you’re starting with Python, or you want to see what a focused, project‑based learning path looks like, you’re in the right place.

## 📌 Why this repo exists

Many learners get stuck in “tutorial hell” – watching videos, reading books, but never writing code alone. I decided to break that cycle.

Every day, I wrote at least one working program.  
No copy‑pasting without understanding.  
No skipping fundamentals.  
Just **coding, breaking, fixing, learning**.

This repo is the evidence.

## 🚀 How to run the code

1. **Clone the repo**  
   ```bash
   git clone https://github.com/DulanDhanush/python_learning.git
   cd python_learning
2. **Make sure Python 3.7+ is installed**
   Check with: 
   ```bash 
   python --version
   ```
3. **Install external dependencies (only for GUI and API projects)**
   ```bash
   pip install pyqt5 requests
   ```
4. **Run any script**
   ```bash
   python calculator.py
   ```
Most scripts (like madlibs.py, quizGame.py) use only the standard library – no installation needed.
